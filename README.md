# Xaero Custom Waypoint Colors
Xaero Custom Waypoint Colors is a mod for Minecraft and an addon to xaero96's minimap mod, allowing for more customization of your waypoints.

It adds the option for an RGB color picker within Xaero's Minimap's waypoint creation & editing system, no longer limiting waypoints to the default 16 preset colors! <br>

**\+ Support for Xaero's World Map**
***
## How to use
When creating a waypoint in Xaero's Minimap, the option to change the waypoint's color will show 16 preset colors plus a new **Custom** option. Clicking the Custom option will bring up an RGB color picker, allowing you to choose any color for the waypoint. 

If you already have a specific color in mind, copy its hex code and paste it into the color picker. <br>

Upon selecting a custom color for a waypoint, you will see in the preset dropdown menu that your color is selected as `Custom:HEXCODE`. If you ever need to change the custom color, you can do so by clicking the same text to open the color picker again.

<br>

Sharing a waypoint with a custom color can be done by using Xaero's built-in **Share** feature. Those who have the mod installed can click the add button for the waypoint in chat and the custom color will be imported.

## Installation
To install the mod, you need the following:
- [Fabric Loader](https://fabricmc.net/use/installer/)
- [Fabric API](https://modrinth.com/mod/fabric-api)
- [Xaero's Minimap](https://modrinth.com/mod/xaeros-minimap)
- [Xaero's Worldmap](https://modrinth.com/mod/xaeros-world-map) (Optional)
